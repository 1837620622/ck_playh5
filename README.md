# 传康云在线游戏平台 (CK PlayH5)

欢迎来到 **传康云在线游戏平台**！这是一个基于 Web 的在线 H5 游戏聚合平台，提供多种类型的休闲游戏，即点即玩，无需下载。

## 🎮 项目简介

本项目汇集了多种类型的 HTML5 游戏，界面简洁，分类清晰。用户可以通过侧边栏快速切换不同类别的游戏，包括益智、动作、射击、消除、休闲和棋牌等。

主要特点：
- **即点即玩**：纯 H5 开发，无需安装任何插件。
- **分类清晰**：侧边栏提供直观的游戏分类导航。
- **响应式设计**：适配不同屏幕尺寸。
- **简洁 UI**：基于 Material Dashboard 风格设计。

## 📂 目录结构

```
.
├── css/                    # 样式文件 (Bootstrap, Material Dashboard, FontAwesome 等)
├── images/                 # 图片资源 (Logo, 背景图等)
├── js/                     # 脚本文件 (jQuery, Bootstrap, Material Design 等)
├── link/                   # 游戏链接及分类页面
│   ├── 0.html              # 全部游戏
│   ├── 1.html              # 益智游戏
│   ├── 2.html              # 动作游戏
│   ├── ...                 # 其他分类
│   └── index.html          # 默认欢迎页或主控页
├── yxmb/                   # 游戏资源文件 (各类游戏的具体代码和素材)
├── index.html              # 主入口文件
└── README.md               # 项目说明文档
```

## 🚀 部署与运行

### 本地运行
1. 克隆项目到本地：
   ```bash
   git clone https://github.com/1837620622/ck_playh5.git
   ```
2. 直接双击打开 `index.html` 即可在浏览器中预览。
   或者使用任何静态文件服务器（如 VS Code 的 Live Server, Python SimpleHTTPServer 等）运行。

### 部署建议
本项目为纯静态网页，可以部署在任何静态托管服务上：
- GitHub Pages
- Vercel
- Netlify
- Nginx / Apache 服务器
